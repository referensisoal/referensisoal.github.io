The correct sentences is I did not do English assignment from Mrs. Annisa.

Explanation

In constructing a sentence, the first thing we determine is to find the subject first then followed by the object and predicate of a sentence.

I => Subject

Did not do English assignment => Object

From Mrs. Annisa => Predicate

The meaning of the sentences is,

saya tidak mengerjakan tugas bahasa Inggris yang diberikan oleh Guru Annisa.

Conclusion

So, if we are done finding the subject, object and predicate, we got the complete sentences. The complete sentences is I did not do English assignment from Mrs. Annisa.
