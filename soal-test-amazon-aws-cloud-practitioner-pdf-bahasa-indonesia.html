8. Layanan mana yang dapat meninjau detail aktivitas pengguna dan panggilan API dalam lingkungan AWS Anda? A AWS CloudTrail B Amazon CloudWatch C AWS Trusted Advisor D Amazon Inspector


Jawaban:

A. AWS CloudTrail

Penjelasan:

AWS CloudTrail adalah layanan yang mencatat aktivitas pengguna dan panggilan API dalam lingkungan AWS Anda. CloudTrail mengumpulkan informasi tentang aktivitas pengguna, seperti akses ke sumber daya AWS, dan panggilan API, seperti membuat, mengubah, atau menghapus sumber daya AWS. Informasi ini disimpan dalam file log yang disebut "trail" yang dapat Anda gunakan untuk meninjau aktivitas pengguna dan panggilan API dalam lingkungan AWS Anda.

Amazon CloudWatch adalah layanan yang memungkinkan Anda untuk mengatur, mengukur, dan memonitor performa sistem dan aplikasi di lingkungan AWS Anda. CloudWatch memungkinkan Anda untuk mengumpulkan dan memonitor data kinerja, seperti pemakaian CPU, latensi aplikasi, dan error aplikasi.

AWS Trusted Advisor adalah layanan yang memberikan saran dan rekomendasi untuk meningkatkan efisiensi, keamanan, dan kehandalan aplikasi di lingkungan AWS Anda. Trusted Advisor memberikan saran tentang masalah seperti penggunaan sumber daya yang tidak optimal, pengaturan keamanan yang tidak aman, dan performa yang tidak optimal.

Amazon Inspector adalah layanan yang memungkinkan Anda untuk memeriksa aplikasi web untuk kerentanan keamanan dan konfigurasi tidak sesuai dengan standar industri. Inspector menggunakan teknik otomatisasi dan analisis yang sama yang digunakan oleh tim keamanan profesional untuk menemukan kerentanan dan memberikan rekomendasi untuk memperbaikinya.

Jawaban:

Jawabannya adalah Mengirimkan konten ke pelanggan melalui jaringan global edge locations.

Penjelasan:

Amazon CloudFront adalah layanan Content Delivery Network (CDN) yang memungkinkan mengirimkan konten ke pelanggan melalui jaringan global edge locations. Edge locations adalah lokasi di seluruh dunia di mana Amazon CloudFront menyimpan salinan konten yang kirimkan.

Ketika pelanggan mengakses konten, CloudFront mengirimkan salinan konten terdekat dari edge location yang terdekat dengan pelanggan, sehingga memastikan bahwa konten tersedia dengan cepat dan meminimalkan latensi.

Jawaban:

Belajar Literasi di Makam RA Kartini

KOTA – Gerakan penguatan pendidikan karakter atau PPK terus dikembangkan SMPN 2 Rembang. Kemarin, sebanyak 40 siswa-siswinya diajak membuka cakrawala budaya literasi dengan metode pembelajaran outdoor kunjungan di makam dan Museum RA Kartini. Salah Satu Pokja PPK dari SMPN 2 Rembang Nurhadi menuturkan, makam dan museum RA Kartini dikunjungi dalam rangka menanamkan nilai nasionalisme. Sebab, sebagian  masyarakat  ada yang tidak melihat peninggalan sejarah. Bahkan nyaris dilupakan.

”Makanya butuh pemantapan dan pemahaman. Sekaligus mereka harus tahu peninggalan-peninggalan sejarah,” ungkapnya kepada Jawa Pos Radar Kudus kemarin (16/11).

Nurhadi menyebutkan, pembelajaran outdoor tidak hanya sekadar berkunjung. Namun di Makam RA Kartini Bulu siswa bersama pembimbing menggelar doa dan tahlil. Tahlil sendiri dipimpin secara langsung guru agama terkait.”Kemudian dilanjutkan melihat silsilah bersama juru kuncinya,” terangnya.  Pihaknya berharap pada pelajar dapat mengetahui sejarah Kabupaten Rembang. Termasuk nama-nama keluarga dari Bupati Rembang pertama. Dilanjutkan anak-anak dengan dipandu guru melakukan wawancara untuk membuat laporan. Kepala sekolah SMPN 2 Rembang Lilik Murdiatno mengapresiasi kegiatan literasi dalam rangka meningkatkan PPK. Lebih-lebih sekolahnya sekaligus menjadi pilot penguatan karakter di kabupaten Rembang.

”Memang banyak sekali kegiatannya. Salah satunya pembelajaran outdoor di makam RA Kartini,” ungkapnya.

40. Komponen atau layanan mana yang dapat membuat koneksi terdedikasi antara data center Anda dan virtual private cloud (VPC)? A Internet gateway B Virtual private gateway C Amazon CloudFront D AWS Direct Connect


Jawaban:

D. AWS Direct Connect

Penjelasan:

WS Direct Connect adalah layanan yang memungkinkan Anda untuk menghubungkan data center Anda secara langsung ke VPC Anda di AWS dengan menggunakan koneksi terdedikasi. Ini dapat membantu mengurangi latensi dan meningkatkan kinerja aplikasi Anda dengan menghindari ketergantungan pada koneksi internet umum.

Internet gateway (jawaban A) adalah komponen yang digunakan untuk menghubungkan VPC Anda ke internet, sementara virtual private gateway (jawaban B) adalah komponen yang digunakan untuk menghubungkan VPC Anda ke jaringan virtual privat lainnya menggunakan jalur terdedikasi. Amazon CloudFront (jawaban C) adalah layanan CDN (Content Delivery Network) yang memungkinkan Anda untuk menyediakan konten seperti gambar, video, dan file statis kepada pengguna di seluruh dunia dengan latensi yang lebih rendah dan tingkat availabilitas yang tinggi.
