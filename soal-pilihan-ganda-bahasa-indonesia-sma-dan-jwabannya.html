27. bantuin ath cara sma jwabannya,,​


L1 : x^2 + y^2 - 4x - 2y - 11 = 0

L2 : x^2 + y^2 + 20x - 12y + 11 = 0

M1 (-1/2(-4) , -1/2(-2))

M1 (2 , 1)

r1 = V(2^2 + 1^2 + 11)

r1 = V(4 + 1 + 11)

r1 = V16

r1 = 4

M2 (-1/2(20) , -1/2(-12))

M2 (-10 , 6)

r2 = V((-10)^2 + 6^2 - 11)

r2 = V(100 + 36 - 11)

r2 = V125

r2 = 11,18

|M1 M2| = V[(-10 - 2)^2 + (6 - 1)^2]

|M1 M2| = V[(-12)^2 + 5^2]

|M1 M2| = V(144 + 25)

|M1 M2| = V169

|M1 M2| = 13

r1 - r2

= 4 - 11,18

= -7,18

r1 + r2

= 4 + 11,18

= 15,18

(r1 - r2) < |M1 M2| < (r1 + r2)

Maka L1 berpotongan dengan L2
