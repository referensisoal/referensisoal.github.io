Rumus = If + Simple Present, will + verb 1
Conditional Sentence (kalimat pengandaian) merupakan kalimat yang menyatakan tindakan atau peristiwa yang terjadi jika syaratnya terpenuhi. Conditional sentence digunakan untuk menyatakan suatu peristiwa atau tindakan yang mungkin terjadi.

Contoh :
1. If you leave your sister alone, she will sad and miss you (Jika kamu meninggalkan saudara perempuanmu sendirian, dia akan sedih dan merindukanmu).
2. If you give me money, I will help you (Jika kamu memberiku uang, saya akan membantumu).
3. If our football team wins, we will very happy and get presents from our family (Jika tim futsal kami menang dalam pertandingan, kami akan sangat senang dan mendapat hadiah-hadiah dari keluar kami).
4. If I forgot to do my homework, I will be punished (Jika saya lupa membuat PR, saya akan dihukum).
5. If I will get presents if my test got 100, I will study very hard (Jika saya mendapat hadiah jika ulanganku dapat 100, saya akan sangat sering belajar).

21. if i__(kaya). i would travel around the worldtolong jawab pertanyaan condisional sentence diatas ​


Jawaban:If I were rich, I would travel around the world. PEMBAHASAN :

CONDITIONAL SENTENCES

➡ Adalah suatu kalimat pengandaian yang hanya akan terjadi jika syaratnya terpenuhi. Conditional sentences terbagi menjadi 3 tipe :

Conditional type 1Conditional type 2Conditional type 3

Conditional Type 1

Rumus :

If + Simple present, conditional (will + V1)

If clause nya menggunakan simple present, dan conditional sentences nya menggunakan will + V1.

Contoh :

If I have enough money, I will buy some books.

Conditional Type 2

Rumus :

If + simple past, conditional (would + V1)

If clause nya menggunakan simple past, dan conditional sentences nya menggunakan would + V1.

Contoh :

If I had enough money, I would buy some books.

Conditional Type 3

Rumus :

If + past perfect, conditional (would have + V3)

If clause nya menggunakan past perfect, sedangkan conditional sentences nya menggunakan would have + V3.

Contoh :

If I had had enough money, I would have bought some books.

Kalimat soal di atas merupakan kalimat conditional type 2, hal ini dikarenakan :

➡ conditional sentences nya menggunakan would + V1

I would travel around the world.

Maka untuk kalimat if clausenya harus menggunakan simple past. Karena kalimat if clause nya bentuk nominal sentences, dimana terdapat adjective "kaya = rich". Maka harus menggunakan to be. Dan to be untuk conditional sentences bentuk past adalah "were". Jadi untuk melengkapi kalimat di atas adalah..
