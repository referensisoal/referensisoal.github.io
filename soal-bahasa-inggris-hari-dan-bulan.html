Future Tense :

Digunakan untuk menyatakan suatu yang akan terjadi di masa depan.

Future Tense dibagi menjadi dua :

willto be going to

Form :

will :

(+) S + will + V1 + O/C(-) S + will + not + V1 + O/C(?) Will + S + V1 + O/C + ?

to be going to :

(+) S + to be + going to + V1 + O/C(-) S + to be + not + going to + V1 + O/C(?) to be + S + going to + V1 + O/C + ?

Keterangan :

S : Subject (Subjek) ; I, you, they, we, she, he, it, etc.V1 : Infinitive, e. g : go, ride, sleep, eat, drink, walk, etcO : Object (Objek)C : Compliment (Keterangan)to be : is, am, are

Note :

Gunakan to be am untuk subjek I.Gunakan to be is untuk subjek she, he, it, dan benda/objek tunggal, misal : Dog, Lina, Rani, Father, etc.Gunakan subjek are untuk subjek you, they, we, dan benda/objek jamak, misal : you and I, Rina and Lani, etc.
