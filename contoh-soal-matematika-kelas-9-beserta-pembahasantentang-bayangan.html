a. [tex]9^{-3}[/tex]

c. [tex]216^{-1}[/tex] atau jika lebih sederhana [tex]6^{-3}[/tex]

d. [tex]8.625^{-1}[/tex] atau jika lebih sederhana [tex]2^3.5^{-4}[/tex]

e. [tex]2^-1.a^{-2}.b^{-3}[/tex]

f. [tex]a^{-1} + ab^{-5}[/tex]

g. [tex]5x^{-3} + 5.3^{-4}x[/tex]

h. [tex]2^{-1}m^{-3}n^{-2}[/tex]

Penjelasan dengan langkah-langkah:

contoh pangkat negatif [tex]x^{-3}[/tex] = [tex]\frac{1}{x^{3}}[/tex]

a.

[tex]\frac{1}{9^3} = 1.9^-^3 = 9^-^3[/tex]

b.

[tex]\frac{4}{12^{7}} \\= 4.12^{-7} \\= 2^2.(3.4)^{-7} \\= 2^2.(3.2^2)^{-7} \\= 2^2.3^{-7}.2^{-14} \\= 3^{-7}.2^{2+(-14)} \\= 3^{-7}.2^{-12}[/tex]

c.

[tex]\frac{1}{216} = 216^{-1} = (6^3)^{-1} = 6^{-3}[/tex]

d.

[tex]\frac{8}{625} = \frac{2^3}{25.25} = \frac{2^3}{5^2.5^2} = 2^3.5^{-2}.5^{-2} = 2^3.5^{-4}[/tex]

e.

[tex]\frac{4a}{(2ab)^3} \\= \frac{4.a}{2^3.a^3.b^3} \\= \frac{2^2.a}{2^3.a^3.b^3}\\= 2^{2-3}.a^{1-3}.b^{-3}\\= 2^-1.a^{-2}.b^{-3}[/tex]

f.

[tex]\frac{a}{a^2 + b^5} = a(a^2 + b^5)^{-1} = a(a^{2.-1} + b^{5.-1}) = a(a^{-2} + b^{-5}) = a.a^{-2} + a.b^{-5} = a^{1+(-2)} + ab^{-5} = a^{-1} + ab^{-5}[/tex]

g.

[tex]\frac{5x}{(x + 3)^4} = 5x(x+3)^{-4} = 5x(x^{-4} + 3^{-4}) = 5x.x^{-4} + 5x.3^{-4} = 5x^{-3} + 5.3^{-4}x[/tex]

h.

[tex]\frac{4m^3n}{(2m^2n)^3} = \frac{2^2m^3n}{2^3m^6n^3} = 2^{2-3}m^{3-6}n^{1-3} = 2^{-1}m^{-3}n^{-2}[/tex]

25. matematika kelas 9 tolong beserta penjelasannya​


Jawaban:

(27)^4/3 - (25)^3/2 + (32)^2/5

(3^3)^4/3 - (5^2)^3/2 + (2^5)^2/5

3^4 - 5^3 + 2^2

81 - 125 + 4

-40

Jawab: D. -40

Penjelasan dengan langkah-langkah:

[tex]27^{\frac{4}{3} } -25^{\frac{3}{2} } +32^{\frac{2}{5} }[/tex]

Ubahlah bilangan pokok menjadi angka berpangkat seperti 27 menjadi 3³, 25 menjadi 5², dan 32 menjadi 2⁵ sehingga menjadi seperti :

[tex]3^{3\frac{4}{3} } -5^{2\frac{3}{2} } +2^{5\frac{2}{5} }[/tex]

Lalu bilangan pangkat tersebut kita bayangkan seperti perkalian dan bisa kita coret antara pangkat dengan pangkat menjadi :

[tex]3^{4} -5^{3} +2^{2}[/tex]

sehingga akan lebih mudah mencari hasilnya :

81 - 125 + 4 = -40
