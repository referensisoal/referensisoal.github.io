4. soal sbm tolong dijawab cepat nomor 4 yaa.


pencerminan terhadap x = a

............x=a
(s, t) ----------> (2a - s, t)

pencerminan terhadap y = b

.....................y=b
(2a - s, t) ---------------> (2a - s, 2b - t)

garis melalui (s, t) dan (2a - s, 2b - t)

persamaan garis =
(y - y1)/(y2 - y1) = (x - x1)/(x2 - x1)
(y - t)/ (2b - t - t) = (x - s)/(2a - s - s)
(y - t)/(2b - 2t) = (x - s)/(2a - 2s)

garis melalui (0, 0)
(0 - t)/(2b - 2t) = (0 - s)/(2a - 2s)
-t/(2b - 2t) = -s/(2a - 2s)
-t(2a - 2s) = -s(2b - 2t)
-2at + 2st = -2bs + 2st
-2at + 2bs = 2st - 2st
-2at + 2bs = 0
2at = 2bs
a/b = 2s/2t
a/b = s/t
a : b = s : t

= A.

18. tolong bantu jawab ya! susah banget ini soal sbm


.

P(x) = ax⁵ + x⁴ + bx³ + x² + cx + d

dibagi (x + 1)
P(-1) = 3
-a + 1 - b + 1 - c + d = 3
-a - b - c + d = 3 - 2 = 1
-(a + b + c) + d = 1
-(a + b + c) = 1 - d
a + b + c = d - 1

dibagi (x - 1)
P(1) = -7
a + 1 + b + 1 + c + d = -7
d - 1 + 2 + d = -7
2d = -7 - 1
d = -8/2
d = -4 ✔

P(x) dibagi x → x = 0
P(0) = 0 + 0 + 0 + d
P(0) = d
P(0) = -4 → jawaban
