14. soal ujian terdiri dari 6 soal paket A dan 4 soal dari paket B.setiap siswa wajib mengerjakan 8 soal yang terdiri dari 5 soal paket A dan 3 soal paket B.jika soal nomer 1 dan 2 di paket A wajib dikerjakan .tentukan banyaknya cara siswa mengerjakan soal ujian!​


Banyak cara siswa mengerjakan soal ujian adalah 16 cara. Nilai tersebut diperoleh dari perhitungan kombinasi pada setiap paket soal. Simak pembahasan berikut.

Pembahasan

Diketahui:

Soal ujian paket A = 6 soal

Soal ujian paket B = 4 soal

Satu siswa wajib mengerjakan 8 soal dengan rincian:

soal ujian paket A = 5 soal

soal ujian paket B = 3 soal

soal nomor 1 dan 2 paket A wajib dikerjakan

Ditanya: banyak cara siswa mengerjakan soal ujian

Jawab:

Banyak cara siswa mengerjakan soal ujian = banyak cara siwa mengerjakan  paket soal A dan banyak cara siswa mengerjakan paket soal B.

Karena pilihan siswa secara acak atau bebas, maka akan dihitung kemungkinan pemilihan dengan cara kombinasi. Rumus kombinasi secara umum adalah sebagai berikut:

nCr = [tex]\frac{n!}{(n-r)!r!}[/tex] dengan n > r

Banyak cara siswa mengerjakan soal paket A

soal paket A = 6 soal diambil 5 soal dengan ketentuan soal nomor 1 da 2 wajib dikerjakan. Maka kini seorang siwa dapat memilih soal di paket A dengan rincian sebagai berikut

soal paket A = (6-2) soal dijawab (5-2) soal

soal paket A = 4 soal dijawab 3 soal. Artinya setiap siswa dapat memilih secara acak 3 dari 4 soal untuk dijawab. Maka banyak pilihan siswa untuk menjawab soal A adalah:

soal paket A = ₄C₃

soal paket A = [tex]\frac{4!}{(4-3)!3!}[/tex]

soal paket A = [tex]\frac{4.3!}{1!.3!}[/tex]

soal paket A = [tex]\frac{4.3!}{1.3!}[/tex]

soal paket A = 4 pilihan soal

∴Jadi banyak cara siswa mengerjakan soal paket A adalah sebanyak 4 cara.

Banyak cara siswa mengerjakan soal paket B

soal paket B = 4 soal dijawab 3 soal. Artinya setiap siswa dapat memilih secara acak 3 dari 4 soal untuk dijawab. Maka banyak pilihan siswa untuk menjawab soal B adalah:

soal paket B = ₄C₃

soal paket B = [tex]\frac{4!}{(4-3)!3!}[/tex]

soal paket B = [tex]\frac{4.3!}{1!.3!}[/tex]

soal paket B = [tex]\frac{4.3!}{1.3!}[/tex]

soal paket B = 4 pilihan soal

∴Jadi banyak cara siswa mengerjakan soal paket B adalah sebanyak 4 cara.

Banyak cara siswa mengerjakan soal ujian = banyak cara siwa mengerjakan  paket soal A dan banyak cara siswa mengerjakan paket soal B.

Sehingga, banyak cara mengerjakan soal yg dapat dilakukan adalah :

5C4 x 3C2 = [5!/(4!1!)] x [3!/(2!1!)]

= 5 x 3

= 15
