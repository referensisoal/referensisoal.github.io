1. buat lah contoh dari prefix 'on' sebanyak 10 contoh dalam bahasa inggris


Answers

1. on-board = ada di atas kapal/kereta api/pesawat (adjective)

Contoh kalimat : All the passengers are already on-board the ship.

2. onscreen = di atas layar (televisi, film)  (adjective)

The actors have great onscreen chemistry.

3. onlooker = penonton  (noun)

The onlookers didn't try to help the poor child.

4. oncoming = tiba, datang  (adjective)

He didn't see the oncoming train.

5. onset = serangan, gejala awal suatu penyakit  (adjective)

My grandfather was diagnosed with early onset dementia.

6. onslaught = serangan  (noun)

The store wasn't prepared for the onslaught of customers on Black Friday.

7. online = tersambung ke internet, aktif, tersedia dalam suatu sistem komputer  (adjective)

The system is online and ready to go.

8. onside = tidak off-side (istilah dalam sepakbola)  (adjective)

He wasn't onside so the goal didn't count.

9. onward = bergerak maju  (adverb)

He kept onward despite all the obstacles.

10. onshore = ada di (dekat) tepi laut (adjective) , bergerak dari laut menuju ke darat  (adverb)

The wind shifted onshore.

--------

Explanation

Prefix adalah imbuhan yang diletakkan di awal suatu kata yang dapat mengubah arti kata tersebut.

Prefix on- dalam bahasa Inggris mempunyai arti di atas, di, atau menuju.

Prefix on- dalam bahasa Inggris juga dapat berubah menjadi a- atau an-.

4. apa fungsi match prefix


Match Suffix artinya kata yang diganti tidak berlaku untuk kata yang berada pada awalan kata.

Match Prefix artinya kata yang diganti tidak berlaku untuk kata yang berada pada akhiran kata.

Contoh :

kata "kudaku", jika "ku" diganti dengan " saya" dengan satu spasi diawal tapi Match Prefix dan Match Suffix nya tidak diceklist maka akan menjadi " sayada saya".

kata "kudaku", jika "ku" diganti dengan " saya" dengan satu spasi diawal dan hanya Match Prefix yang diceklist maka akan menjadi " sayadaku".

kata "kudaku", jika "ku" diganti dengan " saya" dengan satu spasi diawal dan hanya Match Suffix yang diceklist maka akan menjadi "kuda saya".

-----------

Prefix is a letter or letters attached to the beginning of a word and modifies its meaning.

replay = The referee gave Liverpool the penalty after watching the replay of the incident.impossible = It is impossible that pigs can fly.semifinal = The team lost the semifinal.tricycle = My baby brother is riding his tricycle.bipedal = Humans are bipedal. They walk on two legs.

-----------

Suffix is a letter or letters attached to the end of a word and modifies its meaning.

teacher = Mr Smith is an English teacher.sickness = The sickness caused him to lose his hearing.happily = She smiles happily.comfortable = this chair is very comfortable.homeless = After he lost his house he became homeless.

-------------

Words which have both prefix and suffix :

disrespectfully = The rude boys always speak disrespectfully to their mother.anticlockwise = Turn the knob anticlockwise to open the door.uncertainly = He uncertainly answered the question.dissatisfied = Were we dissatisfied with the service offered by the hotel.impossibility = The impossibility of the task despairs him.

31. menu progran infix ke prefix dan infix ke postfix menggunakan bahasa c++


#include<bits/stdc++.h>

using namespace std;

bool isOperator(char c)  

{  

   return (!isalpha(c) && !isdigit(c));  

}  

 

int getPriority(char C)  

{  

   if (C == '-' || C == '+')  

       return 1;  

   else if (C == '*' || C == '/')  

       return 2;  

   else if (C == '^')  

       return 3;  

   return 0;  

}  

 

string infixToPostfix(string infix)  

{  

   infix = '(' + infix + ')';  

   int l = infix.size();  

   stack<char> char_stack;  

   string output;  

 

   for (int i = 0; i < l; i++) {  

 

       // If the scanned character is an  

       // operand, add it to output.  

       if (isalpha(infix[i]) || isdigit(infix[i]))  

           output += infix[i];  

 

       // If the scanned character is an  

       // ‘(‘, push it to the stack.  

       else if (infix[i] == '(')  

           char_stack.push('(');  

 

       // If the scanned character is an  

       // ‘)’, pop and output from the stack  

       // until an ‘(‘ is encountered.  

       else if (infix[i] == ')') {  

 

           while (char_stack.top() != '(') {  

               output += char_stack.top();  

               char_stack.pop();  

           }  

 

           // Remove '(' from the stack  

           char_stack.pop();  

       }  

 

       // Operator found  

       else {  

             

           if (isOperator(char_stack.top())) {  

               while (getPriority(infix[i])  

                  <= getPriority(char_stack.top())) {  

                   output += char_stack.top();  

                   char_stack.pop();  

               }  

 

               // Push current Operator on stack  

               char_stack.push(infix[i]);  

           }  

       }  

   }  

   return output;  

}  

 

string infixToPrefix(string infix)  

{  

   int l = infix.size(); //String Length

 

   

   reverse(infix.begin(), infix.end()); //Reverse infix  

 

   // Replace ( with ) and vice versa  

   for (int i = 0; i < l; i++) {  

 

       if (infix[i] == '(') {  

           infix[i] = ')';  

           i++;  

       }  

       else if (infix[i] == ')') {  

           infix[i] = '(';  

           i++;  

       }  

   }  

 

   string prefix = infixToPostfix(infix);  

 

   

   reverse(prefix.begin(), prefix.end());  // Reverse postfix  

 

   return prefix;  

}

int main()

{

char c;

string ax;

do

{

 system("cls");

 printf("Menu\n1. Infix to Postfix\n2. Infix to Prefix\n0. Exit\nChoose: ");

 cin >> c;

 printf("\n");

 switch(c)

 {

  case '1':

   printf("Insert Math Function (Infix to Postfix): ");

   cin >> ax;

   cout << "Result: "<< infixToPostfix(ax)<<endl;

   getchar();

   break;

  case '2':

   printf("Insert Math Function (Infix to Prefix): ");

   cin >> ax;

   cout << "Result: "<<infixToPrefix(ax)<<endl;

   getchar();

   break;

 }

 getchar();

}while(c!='0');

return 0;

}

34. Sebutkan 10 contoh dari prefix root dan sufix


MAKNA DAN CONTOH PREFIX ROOT

1. A- (tidak, tanpa). Contoh kata: Atypical, amoral, atheist.

2. Ante- (sebelum). Contoh kata: Antecedent, anteroom.

3. Anti- (berlawanan). Contoh kata: Antioxidant, antibiotic, antithesis.

4. Dis- (tidak). Contoh kata: Disable, dislike, disobedient

5. Ex- (mantan). Contoh kata: Ex-husband, ex-member, ex-president.

6. Extra- (melebihi). Contoh kata: Extraordinary, extracurricular, extraterritorial.

7. Il-, im-, in-, ir- (tidak, tanpa). Contoh kata: Illegible, impatient, inappropriate, irrelevant.

8. Intra- (di dalam, di antara). Contoh kata: Intracellular, intravenous.

9. Macro- (besar). Contoh kata: Macroeconomics, macrofossil, macroscopic.

10. Mis- (salah). Contoh kata: Misheard, misunderstanding, misspelling.

11. Mono- (satu). Contoh kata: Monoatomic, monolog, monogamy.

12. Non- (tidak, tanpa). Contoh kata: Non-alcoholic, non-fiction, non-formal.

13. Post- (setelah). Contoh kata: Postcolonialism, postgraduate, postwar.

14. Pre- (sebelum). Contoh kata: Preposition, pre-test, prepaid.

15. Sub- (di bawah). Contoh kata: Subconscious, submarine, subway.

16. Un- (tidak). Contoh kata: Unlike, unattractive, unfamous.

17. Al- (semua). Contoh kata: Alone, already.

18. Over- (di atas, di luar, melebihi). Contoh kata: Overflow, overcharge.

19. Un- (tidak, membalikkan peristiwa atau keadaan). Contoh kata: Unkind, untie, undo.

20. Mega- (besar). Contoh kata: Megaphone, megaton.

21. Semi- (setengah, sebagian). Contoh kata: Semiconductor, semester, semi-retired.

            Object e=top.data;

            return e;

            Object e =  top.data;
