28. menu dan fungsi MS world tool-file-edit-layout-view​


Penjelasan:

1. FILE !!

Pada tools di file tab ini, terdapat banyak menu yang memiliki fungsinya masing-masing. Opsi yang disediakan pada file tab ini, yakni:

New Document, berfungsi untuk membuka atau membuat file baru.

Recent, berfungsi untuk membuka file yang sebelumnya telah dikerjakan.

Open, berfungsi untuk membuka file yang tersimpan atau terunggah di folder.

Save/Save As, berfungsi untuk menyimpan file yang baru saja dikerjakan.

Print, berfungsi untuk mencetak dokumen atau laporan yang telah dikerjakan di Microsoft Word.

EDIT !!

Untuk mengedit/mereview sebuah projek yang akan dibuat agar hasilnya maksimal.

view !!

View buttons merupakan tools yang berguna untuk mengubah tampilan dokumen. View buttons ini juga memiliki beberapa tools di dalamnya yang tentu memiliki berbagai macam fungsi, yakni:

Web Layout, yang berguna untuk melihat layout dokumen apabila ditampilkan di laman website.

Full Screen Reading, yang bermanfaat untuk melihat dan juga membaca dokumen dalam ukuran yang maksimal.

Print Layout, yang berguna untuk melihat hasil kerja apabila di cetak.

Outline View, yang bermanfaat untuk melihat hasil kerja yang sesuai dengan urutan dan juga bagannya.

Draft View, yang berguna untuk melihat hasil kerja sesuai dengan konsep, sehingga jika salah bisa cepat dibenarkan.

layout !!

Page layout termasuk salah satu fitur yang terdapat di dalam Microsoft Office, sesuai dengan penjelasan di laman resmi Microsoft Office Word disebutkan bahwa page layout adalah salah satu menu dengan fungsi utama sebagai pengatur atau alat yang digunakan untuk mengatur tampilan pada sebuah dokumen

Fungsi Menu dan Ikon Word

bagian bagian ikon word

Setelah mengetahui nama dan fungsi dari bagian dalam lembar kerja MS Word, kita juga harus belajar lebih spesifik tentang fungsi ikon yang bertaburan disana. Berikut ini kami telah merangkum fungsi dan perintah yang dapat dijalankan melalui menu dan ikon pada Microsoft Word.

1. File

Mengenal Bagian dan Fungsi Menu Serta Ikon Microsoft Word

Pertama adalah menu file, disini terdapat berbagai perintah dasar dan ada juga bagian untuk menyesuaikan pengaturan lembar kerja.

2. Home

Mengenal Bagian dan Fungsi Menu Serta Ikon Microsoft Word

Clipboard : Ribbon group ini terdiri dari beberapa ikon yang umumnya berfungsi untuk mengcopy ataupun memindah elemen dalam dokumen.

3. Insert

Mengenal Bagian dan Fungsi Menu Serta Ikon Microsoft Word

Tab menu ini secara keseluruhan berisi fungsi untuk memasukan objek ke dalam dokumen.

4. Page Layout

Mengenal Bagian dan Fungsi Menu Serta Ikon Microsoft Word

Tab menu ini berisi berbagai ikon yang mempunyai fungsi untuk mengatur dan menyesuaikan segala hal yang berhubungan dengan halaman lembar kerja MS Word.

5. Menu Format

Menu format merupakan perintah yang digunakan untuk mengatur format pada sebuah dokumen yang memuat perintah di sebuah dokumen.

6. Menu Tools

Menu Tools memuat berbagai perintah tools yang diandalakan oleh Ms word seperti Spelling dan Grammar, kemudian translate, pembuatan macro dan lain sebagainya.

7. Menu Table

Menu table merupakan perintah yang dipakai ketika kita sedang bekerja menggunakan table seperti membuat sebuah rincian atau detil pembelian.

8. Menu Windows

Menu windows memuat perintah untuk mengatur tampilan pop jendela microsoft word layaknya seperti New Windows, Arrange All, dan Split. Menu windows dapat berisi daftar file yang sedang dibuka di microsoft word.

9. Menu Help

Menu help berfungsi untuk memuat perintah bantuan jika kamu mengalami permasalahan saat mengakses microsoft word.

38. Bagaiman cara keluar dari MS world adalah


Jawaban:

Untuk keluar dari Microsoft Word, Anda dapat mengikuti salah satu metode berikut ↓

1. Klik pada tombol "Tutup" (X) di sudut kanan atas jendela aplikasi. Ini adalah cara paling umum untuk keluar dari sebagian besar aplikasi di sistem operasi Windows.

2. Gunakan pintasan keyboard. Pada keyboard, tekan dan tahan tombol "Alt" dan kemudian tekan tombol "F4". Ini akan mengaktifkan perintah untuk menutup aplikasi aktif. Jika Word adalah satu-satunya aplikasi yang sedang berjalan, ini akan menutup Word.

3. Jika Anda menggunakan menu versi lama di Word, klik pada "File" di menu atas. Kemudian, pilih "Exit" atau "Close" untuk keluar dari Word.

4. Jika Anda menggunakan stik pita Word yang lebih baru, klik pada tab "File" di pojok kiri atas jendela aplikasi. Kemudian, pilih opsi "Close" atau "Exit" yang biasanya terletak di bawah daftar perintah di sebelah kiri.

Jika Anda telah menyimpan perubahan pada dokumen yang sedang diperbarui, Word akan menanyakan apakah Anda ingin menyimpan perubahan sebelum menutup. Anda dapat memilih untuk menyimpan, tidak menyimpan, atau membatalkan tindakan penutupan.

40. tuliskan 5 langkah membuat diagram di ms world​


Jawaban:

1. Pilihlah sel kosong di mana Anda ingin menempatkan grafik mini.

2. Pilih “Insert” > “Sparkline”. Setelah itu, pilih tipe grafik mini yang ingin Anda buat seperti baris atau kolom.

3. Kemudian, pilih sel di baris atau kolom yang ingin Anda buat grafiknya dan tekan “Ok” setelah selesai.

4. Jika Anda ingin semua data pada tabel memiliki grafik mini, Anda cukup memposisikan kursor pada pojok kiri bawah sel yang telah berisi grafik. Setelah muncul tanda plus (+) hitam, tarik ke bawah. Maka, otomatis semua data Anda akan memiliki grafik mini.

5. Setelah semua grafik mini terbuat, Anda juga dapat mengatur desainnya dengan menekan menu “Design/Desain”. Dalam menu desain Anda dapat memilih warna grafik, bentuk grafik, dan lain sebagainya sesuai kebutuhan Anda.
