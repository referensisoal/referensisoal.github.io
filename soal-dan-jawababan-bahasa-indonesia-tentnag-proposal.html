1/Rp = 1/R1 + 1/R2
1/Rp = 1/6 + 1/12 = 3/12
Rp = 12/3 = 4 ohm

Rs = R3 + Rp = 14 + 4 = 18 ohm

I = E/Rs = 80/18 = 40/9 A

I = I1 + I2

R1 : R2 = 1 : 2

I1 = 1/3 x I = 1/3 x 40/9 = 40/27 A = 1,48 A
